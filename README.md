## Setup

```bash
git clone <repository-url>
cd "Ruby on Rails Test Task"
bundle install
bin/rails db:prepare
```

## Run

```bash
bin/rails server
```

Server starts at http://localhost:3000

## Test

```bash
bundle exec rspec
```

